Exercício proposto do Canal: Curso em Vídeo

1) Criar um script Python que leia o nome de uma pessoa
e mostre uma mensagem de boas-vindas de acordo
com o valor digitado.

2) Criar um script Python que leia o dia, o mês
e o ano de nascimento de uma pessoa e mostre uma mensagem
com a data formatada.

3) Criar um script em Python que leia dois números
e tente mostrar a soma entre eles.
